## Coding Interview Tips
- Understand, think out loud
- Design
- Explain, Communicate, Tell the Plan, Ask Question
- Multi respones, Open-ended
- Collaborate

- The code needs to be efficient
- Common algorithms and data structures
- Outline the algorithm before into coding
	- Improve and refine with interviewer
- No need to come with optimal soultion first
	- Iterate is ok
- Fundamental undesrtanding of basic algos and data structures
	- Sorting, Search, binary search
	- Divide and conquer
	- Dynamic programming and memorization
	- Greedy algorithms
	- Recursions
	- Graph traversal, breadth-first, depth-first
- 1 or 2 test cases
