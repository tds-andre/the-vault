
Em se tratando de IA, uma diferenciação **importante** que você tem que ter em mente é entre **desenvolvedores/pesquisadores**, **operadores/integradores** e usuários:
- pesquisador (e.g., ML Researcher): é quem desenvolve os algorítimos do zero e entende toda a matemática por trás, normalmente para fins acadêmicos (e.g., escrever um paper).
- desenvolvedor (e.g., ML Engineer): é quem utiliza algoritmos semi-prontos e entende toda a matemática por trás a fim de criar um produto para a empresa (é o que eu faço)
- **integrador: é quem utiliza e integra ou "junta" IAs prontas do mercado, para criar um produto para a empresa - é o que está bombando no momento e é oque faz sentido para você (e não precisa entender a matemática)**
- **operador: é quem usa IAs prontas, integradas ou standalone - é o que você faz quando usa ChatGPT ou Copilot para fins profissionais**
